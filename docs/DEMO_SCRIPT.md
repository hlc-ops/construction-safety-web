# 3 分钟演示视频脚本

**总时长**:180 秒(3 分钟)
**目标观众**:面试官 / GitHub 访客 / 潜在合作方
**核心信息**:全栈能力 + AI 双保险 + 商业级功能矩阵
**录制工具推荐**:OBS Studio(免费) + 麦克风 + 1080p 60fps

---

## 录制前准备清单

- [ ] 浏览器 1080p,关闭书签栏和扩展图标
- [ ] 后端启动并确认 `/api/health` 返回 200
- [ ] 前端构建后由 Flask 服务,避免 dev server 的热更新蹦字
- [ ] 预先准备:1 张测试图(含未戴安全帽)、1 个 10 秒测试视频、1 个公网 RTSP 流(或本地摄像头)
- [ ] 数据库里预填 30-50 条历史告警(让报表/驾驶舱有数据)
- [ ] 用 Edge / Chrome 隐身窗口,登录态干净
- [ ] 录屏分辨率 1920×1080,关闭通知

**画面**:停在驾驶舱,鼠标缓慢滑过

---

### 【00:30-00:55】图片检测 + LLM 二次复核(25s)

**画面**:
- 进入"图片检测"页
- 拖拽测试图(预先准备的工地照片)
- YOLO 框出 3 个目标(0.3 秒)
- 点击"LLM 二次复核"按钮
- 右侧展示 LLM 输出的整改建议(打字机效果)

**旁白**:
> "核心是 YOLO 加大模型的双保险:YOLO 快速定位违规——比如未戴安全帽——然后视觉大模型复核场景,给出文字化的整改建议。**这一步过滤了大量误报**。"

**字幕**:`YOLO 一道筛 + LLM 二道审 · 双保险降误报`

---

### 【00:55-01:20】视频检测(25s)

**画面**:
- 进入"视频检测"
- 上传 10 秒测试视频
- 视频播放,canvas 浮层叠加检测框(10 fps 渲染)
- 命中高危时,右上角告警通知弹出 + 蜂鸣声
- 检测结果实时入库

**旁白**:
> "视频检测采用分层渲染——原视频 30fps 流畅播放,AI 检测框独立 10fps 叠加,既不卡顿又能看清告警。命中高危违规自动入库、自动推送。"

**字幕**:`分层渲染 · 30fps 视频 + 10fps AI 浮层`

---

### 【01:20-01:45】RTSP 摄像头 + 电子围栏(25s)

**画面**:
- 进入"摄像头管理"
- 列表显示已添加的摄像头(在线状态)
- 点击一台进入"电子围栏"配置
- 鼠标在画面上点 4 个点画一个多边形
- 保存,提示"围栏已启用,只检测多边形内的违规"

**旁白**:
> "支持 RTSP 网络摄像头,可配置电子围栏——只在指定区域检测,减少干扰画面;同时支持每类违规独立调整置信度阈值。"

**字幕**:`RTSP 接入 · 电子围栏 · 分类灵敏度`

---

### 【01:45-02:05】告警记录 + AI 智能分级(20s)

**画面**:
- 进入"告警记录"
- 列表显示风险列 + AI 紧急度列
- 标签:`高危` `立即处置`(红) / `紧急`(橙) / `一般`(灰)
- 鼠标悬停"立即处置"显示理由 tooltip
- 点击一条记录,详情对话框展开:告警截图、视频片段回放、LLM 给出的紧急度理由

**旁白**:
> "每条高危告警都会被视觉大模型再次评估,判断真实紧急度。**只有立即处置和紧急级别才会推送钉钉/企微**,避免管理员被刷屏。"

**字幕**:`AI 告警分级 · 智能过滤误报推送`
